Skill

  • HTML merupakan markup language dan bukan merupakan sebuah bahasa pemrogramman,html berjalan diatas browser dan berfungsi sebagai kerangka pembangun website

  • Seperti html,CSS bukanlah bahasa pemrogramman,dan CSS juga berjalan diatas web browser,dan berfungsi sebagai pengatur tampilan sebuah web,agar terlihat lebih cantik dan terstruktur.

  • Javascript merupakan sebuah bahasa pemrogramman tingkat tinggi,dan dapat bekerja didalam sebuah website,berfungsi sebagai pengendali bagian dalam dan luar website,agar website terlihat lebih hidup dan menarik.

  • PHP merupakan bahasa pemrogramman kedua yang saya pelajari setelah Javascript,dikarenakan bahasa ini juga berjalan diatas web browser.PHP merupakan bahasa pemrogramman script seperti javascript dan python,dan bisa digunakan secara bersamaan dengan html,dan biasa digunakan untuk membuat sebuah web yang dinamis dan bisa juga diintegrasikan dengan sebuah database untuk menyimpan data dari sebuah website,agar website terlihat dinamis.

  • Bootstrap bukanlah bahasa pemrogramman,Bootstrap merupakan sebuah CSS Framework yang berjalan diatas CSS,yang berfungsi untuk mempermudah dalam membuat sebuah web yang cantik.

  • MySql merupakan sebuah perangkat lunak management basis data(database),yang berfungsi biasanya sebagai database dalam sebuah website,yang bisa diintegrasikan dengan bahasa pemrogramman seperti PHP.
